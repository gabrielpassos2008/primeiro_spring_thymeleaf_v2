Gerenciador de Tarefas – Versão 2

Esta é a evolução do meu primeiro projeto utilizando Spring Boot integrado com Thymeleaf para renderização das páginas web.

Nesta nova versão, o sistema deixou de ser apenas um CRUD básico e passou a incorporar melhorias estruturais, funcionais e visuais, aproximando-se mais de uma aplicação real.

O objetivo desta versão foi aprofundar os conceitos fundamentais do desenvolvimento backend com Java, aplicando de forma mais organizada o padrão de arquitetura MVC (Model-View-Controller), reforçando a separação de responsabilidades entre Controller, Service e Repository, além de melhorar a experiência visual com Bootstrap.

Evoluções da Versão 2

  • Implementação correta do padrão MVC (Controller → Service → Repository)
  • Separação clara de responsabilidades entre camadas
  • Controller deixando de acessar lógica diretamente e usando Service
  • Criação de método de busca de tarefas
  • Implementação de filtros (tarefas pendentes e concluídas)
  • Validação básica de entrada (null e isBlank())
  • Uso de Stream() para manipulação de listas
  • Uso de filter() para aplicar regras de busca
  • Centralização da lógica de contagem de tarefas no backend
  • Home agora recebe dados dinâmicos do Controller
  • Preparação do projeto para futura implementação de tratamento de erros
