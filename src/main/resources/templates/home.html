Meu primeiro Spring usando o thymeleaf!

Este foi o meu primeiro projeto utilizando Spring Boot integrado com Thymeleaf para renderização das páginas web.

O objetivo deste sistema é praticar os conceitos fundamentais do desenvolvimento backend com Java, aplicando o padrão de arquitetura MVC (Model-View-Controller), organização em camadas e integração entre controller, service e repository.

Durante o desenvolvimento, trabalhei com:

  • Estrutura de rotas com Spring
  • Templates dinâmicos com Thymeleaf
  • Integração com Bootstrap para estilização
  • Manipulação de objetos e listas
  • Conceitos de orientação a objetos
  • Estruturação de projeto seguindo boas práticas
